# 📊 Pandas: Series & DataFrame

## 🔹 Series

* 1D data structure (single column)
* Contains values + index
* Used for simple, linear data

## 🔹 DataFrame

* 2D data structure (table)
* Contains rows + columns
* Used for structured datasets
* performed operations on them 

## 🔹 Key Difference

* Series → single column
* DataFrame → full table

## 🔥 Summary

👉 Series = 1D
👉 DataFrame = 2D
